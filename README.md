2. **Download the Script**

3. **Run the Script**  
   Open your terminal or command line and use the following commands to run the tool:
   ```bash
   wget https://raw.githubusercontent.com/Cd1s/network-latency-tester/main/latency.sh
   chmod +x latency.sh
   ./latency.sh
   ```
